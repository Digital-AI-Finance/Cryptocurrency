# Course Assessments

This folder contains the assessment materials for the cryptocurrency course.

## Assessment Structure

The instructor provides:
- Final project report template (see below)
- Quizzes are available in the `quiz/` directory (4 parts per lesson, 20 questions each)

## Final Project

Students create their own ERC-20 token with:
1. Custom name and symbol
2. Defined total supply
3. At least one custom feature
4. Local deployment and testing

## Submission Requirements

See `final_report_template.md` for the final report format.
